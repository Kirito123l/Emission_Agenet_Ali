## 一、目录总览

该目录是 **Emission Agent** 项目的主目录，这是一个面向研究的 LLM + 工具调用（Tool-Use）车辆排放分析系统。它整合了 FastAPI 后端、聊天式 Web UI、领域计算工具、基于文件的流程、知识/RAG 检索、GIS 结果可视化以及工程评估框架。

### 2.11 `outputs/` — 输出结果（含 Excel 表格）

**该目录包含 3123 个文件**，主要是系统运行后生成的排放计算结果 Excel 文件。

#### Excel 表格内容详解

所有 Excel 文件采用统一的命名格式：
```
{session_id}_input_emission_results_{YYYYMMDD}_{HHMMSS}.xlsx
```

**表格结构（以 `04d8320e_input_emission_results_20260412_220654.xlsx` 为例）：**

| 列名 | 数据类型 | 说明 |
|------|----------|------|
| `link_id` | 整数 | 路段唯一标识 |
| `link_length_km` | 浮点数 | 路段长度（公里） |
| `traffic_flow_vph` | 整数 | 小时交通流量（辆/小时） |
| `avg_speed_kph` | 整数 | 平均车速（公里/小时） |
| `fleet_composition` | JSON 字符串 | 车队构成（含各车型百分比和小时车流量） |
| `emissions_by_vehicle` | JSON 字符串 | 按车型细分的排放量（CO2、NOx 等） |
| `total_emissions_kg_per_hr` | JSON 字符串 | 每小时总排放量（kg/hr） |
| `emission_rates_g_per_veh_km` | JSON 字符串 | 单车公里排放率（g/veh·km） |
| `CO2_kg_h` | 浮点数 | 每小时 CO2 排放量（kg） |
| `NOx_kg_h` | 浮点数 | 每小时 NOx 排放量（kg） |

**车队构成（fleet_composition）示例：**
```json
{
  "Passenger Car": {"source_type_id": 21, "percentage": 70.0, "vehicles_per_hour": 630.0},
  "Passenger Truck": {"source_type_id": 31, "percentage": 20.0, "vehicles_per_hour": 180.0},
  "Light Commercial Truck": {"source_type_id": 32, "percentage": 5.0, "vehicles_per_hour": 45.0},
  "Transit Bus": {"source_type_id": 42, "percentage": 3.0, "vehicles_per_hour": 27.0},
  "Combination Long-haul Truck": {"source_type_id": 62, "percentage": 2.0, "vehicles_per_hour": 18.0}
}
```

**按车型排放量（emissions_by_vehicle）示例：**
```json
{
  "Passenger Car": {"CO2": 11.8226, "NOx": 0.0002},
  "Passenger Truck": {"CO2": 4.4564, "NOx": 0.0002}
}
```

**文件规模示例：**
- 小型结果：约 125 行（如 `04d8320e_input...xlsx`）
- 大型结果：约 25,370 行（如 `06417474_input...xlsx`），覆盖完整路网

## 四、系统工作流程

1. 用户通过 Web UI / CLI / API 发送消息
2. `core/router.py` 统一路由分析意图
3. `core/assembler.py` 组装上下文（记忆 + 文件 + 历史）
4. LLM 决定调用哪些工具
5. `core/executor.py` 执行工具（含参数标准化）
6. 工具调用 `calculators/` 或 `services/` 完成计算
7. 结果返回给 LLM 综合生成回复
8. 回复通过 `api/` 返回给用户，可包含图表/表格/文件/GIS 地图

---

```
用户输入 → api/routes.py → core/router.py → core/assembler.py
                                              ↓
                                        LLM 决策
                                              ↓
                              tools/ → calculators/ → services/
                                              ↓
                                    结果 → outputs/*.xlsx
                                              ↓
                                    返回用户（文本/图表/地图）
```
