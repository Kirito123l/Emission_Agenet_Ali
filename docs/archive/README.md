# Documentation Archive

Files under `docs/archive/` are retained for traceability. They may preserve
legacy plans, historical reports, paper outlines, prompts, or old implementation
notes that are useful for audit history.

Archived files are not active implementation or design sources. The active v1
architecture baseline is:

- `docs/architecture/emissionagent_v1_architecture_freeze.md`

Archived scripts are not part of the current evaluation pipeline unless they are
explicitly restored and revalidated in a future phase.
