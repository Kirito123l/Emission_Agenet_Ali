# 预处理的 GIS 数据

本目录包含预处理的 GeoJSON 文件，用于地图可视化。

## 文件说明

- **basemap.geojson** (67.6 KB)
  - 上海市行政区划底图
  - 包含16个行政区的多边形边界
  - 几何简化容差: 0.001 度
  - 保留字段: name, geometry

- **roadnetwork.geojson** (5.9 MB)
  - 上海市路网底图
  - 简化后的道路网络
  - 几何简化容差: 0.0005 度
  - 保留字段: geometry, highway, name (如果存在)

## 优势

1. **无需实时处理**: 服务器直接读取预处理的 GeoJSON，无需实时读取 Shapefile
2. **减少依赖**: 服务器不需要安装 geopandas 和 fiona
3. **更快响应**: 文件已简化和优化，加载速度更快
4. **易于部署**: 纯文本 JSON 格式，易于版本控制和部署

## 重新生成

如果需要重新生成这些文件（例如更新源数据），运行：

```bash
python preprocess_gis.py
```

该脚本会：
1. 读取 `GIS文件/` 目录下的 Shapefile
2. 简化几何以减小文件大小
3. 只保留必要字段
4. 输出到 `static_gis/` 目录

## 注意事项

- 几何简化会损失一些细节，但对于地图底图来说足够精确
- 如果需要更高精度，可以调整 `preprocess_gis.py` 中的 `simplify()` 参数
- 路网文件较大 (5.9 MB)，如果需要进一步减小，可以增加简化容差或过滤掉次要道路
