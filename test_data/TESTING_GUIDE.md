# 测试指南

## 快速开始

### 1. 验证测试文件
```bash
cd test_data
python3 verify_test_files.py
```

### 2. 启动服务器
```bash
PORT=8001 python3 run_api.py
```

### 3. 打开浏览器
访问: http://localhost:8001

## 地图可视化验证清单

使用 `test_6links.xlsx` 或 `test_20links.xlsx` 测试时，检查以下地图特性：

### 基础样式
- [ ] 白色背景（无在线瓦片地图）
- [ ] 浅灰色行政区划边界线
- [ ] 极浅灰色行政区划填充
- [ ] Attribution 显示 "© Emission Agent"

### 路段显示
- [ ] 路段呈现弯曲形状（不是直线）
- [ ] 颜色编码清晰（绿→黄→红）
- [ ] 线宽适中（weight: 4）
- [ ] 透明度合适（opacity: 0.8）

### 交互功能
- [ ] 点击路段显示 popup
- [ ] Popup 包含：link_id, 排放量, 单位排放率, 速度, 流量, 长度
- [ ] 可以切换污染物类型
- [ ] 地图缩放和平移流畅

### 图层控制
- [ ] 右上角有图层控制按钮
- [ ] 可以切换"路网"图层
- [ ] 路网图层默认隐藏
- [ ] 路网图层显示时为浅灰色细线

### 降级方案
如果 GIS 底图文件不存在：
- [ ] 自动回退到 CartoDB Positron 瓦片底图
- [ ] 控制台显示警告信息
- [ ] 地图功能正常

## 常见问题排查

### 问题1：地图不显示
**可能原因**:
- 没有几何信息（预期行为）
- Leaflet 库未加载
- 地图容器未正确初始化

**检查**:
```javascript
// 打开浏览器控制台
console.log(typeof L); // 应该是 "object"
```

### 问题2：路段显示为直线
**可能原因**:
- 使用了只有2个坐标点的测试数据
- 几何信息解析失败

**检查**:
- 使用 `test_6links.xlsx` 或 `test_20links.xlsx`（包含多点几何）
- 查看控制台是否有几何解析错误

### 问题3：没有下载按钮
**可能原因**:
- download_file 生成失败
- 后端错误

**检查**:
- 查看服务器日志
- 检查 `outputs/` 目录是否有生成的文件

### 问题4：地图背景是 CartoDB 瓦片
**可能原因**:
- GIS 底图文件不存在（降级方案）
- GIS API 返回错误

**检查**:
```bash
# 检查 GIS 文件是否存在
ls -la "GIS文件/上海市底图/上海市.shp"
ls -la "GIS文件/上海市路网/opt_link.shp"
```
