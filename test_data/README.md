# 测试数据说明

本目录包含用于测试排放计算和地图可视化功能的测试文件。所有数据基于上海市真实路网，包含多点几何信息以展示弯曲道路效果。

## 文件列表

### 1. test_6links.xlsx
- **格式**: Excel (.xlsx)
- **路段数**: 6条
- **列**: link_id, length (km), flow (辆/小时), speed (km/h), geometry (WKT格式)
- **坐标点数**: 平均14.3个点/路段 (范围: 5-26个点)
- **用途**: 测试基本的排放计算和地图可视化功能

### 2. test_6links.zip
- **格式**: Shapefile (压缩包)
- **路段数**: 6条
- **包含文件**: .shp, .shx, .dbf, .prj, .cpg
- **用途**: 测试 Shapefile 格式上传和处理

### 3. test_20links.xlsx

### 4. test_20links.zip

## 数据来源

所有测试数据从 `GIS文件/上海市路网/opt_link.shp` 中抽取，优先选择：
- 市中心区域（经度 121.4-121.5，纬度 31.2-31.3）
- 坐标点数 ≥5 的路段（展示弯曲道路效果）

交通参数（flow 和 speed）为随机生成：
- flow: 500-2000 辆/小时
- speed: 30-80 km/h

## 地图可视化效果

使用这些测试文件，地图应该展示：
- ✅ 白色干净背景（无在线瓦片底图）
- ✅ 浅灰色行政区划边界
- ✅ 弯曲的道路路段（不是直线）
- ✅ 颜色编码的排放强度（绿→黄→红）
- ✅ 可选的路网图层
- ✅ 学术论文风格的专业可视化

## 注意事项

1. **坐标系**: 所有几何数据使用 WGS84 坐标系 (EPSG:4326)
2. **单位**:
   - length: 公里 (km)
   - flow: 辆/小时 (veh/h)
   - speed: 公里/小时 (km/h)
3. **几何格式**:
   - Excel: WKT LINESTRING 格式
   - Shapefile: 原生几何对象
