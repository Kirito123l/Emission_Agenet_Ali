# 上海市完整路网测试指南

## 文件说明

**test_shanghai_full.xlsx** / **test_shanghai_full.zip**

这是基于上海市真实路网数据生成的大规模测试文件，用于验证系统在处理完整城市路网时的可视化效果和性能。

### 数据规模
- **路段数**: 150条
- **覆盖范围**: 上海市中心核心区域
  - 经度: 121.34° ~ 121.56°
  - 纬度: 31.14° ~ 31.35°
  - 中心点: (121.45°, 31.25°)
- **坐标点数**: 平均9.5个点/路段 (范围: 3-47个点)
- **路段类型**:
  - 主干道 (primary): 66条
  - 次干道 (secondary): 38条
  - 快速路 (trunk): 17条
  - 快速路匝道 (trunk_link): 14条
  - 高速公路匝道 (motorway_link): 6条
  - 其他: 9条

## 常见问题

### Q1: 地图加载很慢
**可能原因**:
- 150条路段的几何数据较大
- 浏览器性能限制

**解决方案**:
- 使用现代浏览器（Chrome/Edge/Firefox 最新版）
- 关闭其他占用资源的标签页
- 检查网络连接（如果使用在线瓦片底图）

### Q2: 路段颜色区分不明显
**可能原因**:
- 150条路段密度较高
- 排放值分布不均匀

**解决方案**:
- 放大地图查看局部区域
- 使用 popup 查看具体数值
- 切换不同污染物类型对比

### Q3: 地图显示不完整
**可能原因**:
- 地图容器尺寸问题
- 几何数据解析错误

**解决方案**:
- 刷新页面重新加载
- 检查浏览器控制台错误信息
- 尝试使用 Shapefile 格式 (test_shanghai_full.zip)
